## 2. 収集する情報

SSW Compass は **個人情報を収集しません**。

サービスの性質上、以下は**受け付けません**:
- 在留カード番号・パスポート番号・マイナンバー
- 氏名・生年月日（年月のみ可）・住所
- その他個人を特定できる情報

入力検知時: 自動的にブロックし、処理しません。

**取り扱う運用メタデータ** (Google Cloud Logging):
- サービスへのアクセス日時およびセキュリティメタデータ (IP アドレスは Cloud Run / Cloud Armor のセキュリティログで処理される場合がありますが、プロファイリングには使用しません)
- ツール呼び出し種別 (個人情報なし)
- エラーログ

---

## 3. 情報の利用目的

- サービスの安定運用と品質改善
- 不正アクセス・不正利用の検知と防止
- 法令・監査要件に基づく記録の保持

---

## 4. 情報の保管と保持期間

- 個人情報は保存しません。ビザ申請内容や個人識別子をサーバーに保存しません。
- 監査ログは Cloud Logging から GCS の WORM ストレージに転送し、**7 年間保持**します（改ざん防止・追記専用。ADR-015）。監査ログには個人情報を含めません。
- 運用メタデータは目的達成に必要な期間のみ保持し、その後は削除します。

---

## 5. 情報の第三者提供

収集した運用メタデータを第三者に販売・提供することはありません。個人情報は保有しないため第三者と共有しません。
以下の場合を除きます:
- 法令に基づく開示請求
- 生命・身体・財産の保護に必要な場合

---

## 6. データの所在と越境移転

| サービス | 用途 | データ所在 |
|---|---|---|
| Google Cloud Run | アプリケーション実行 | 日本 (asia-northeast1) |
| Google Cloud Logging | ログ収集 | 日本 |
| Google Cloud Armor | セキュリティ | グローバル |
| Vertex AI Search | 情報検索 | 日本 |

SSW Compass は個人情報を意図的に収集しません。運用上のセキュリティメタデータは Cloud Armor 等の Google Cloud グローバル edge サービスで処理される場合がありますが、ビザ申請内容や個人識別子を越境移転しません。

---

## 7. 利用者の権利

本サービスは匿名・無料の情報提供であり、原則として個人情報を保有しません。そのため特定個人に紐づく開示・訂正・削除の対象データは通常存在しません。プライバシーに関するご質問・ご要望は a_kabe@sugu-kuru.co.jp へご連絡ください。適用法令に基づき適切に対応します。

> **行政書士レビュー要**: 個人情報保護法 (APPI) 上の請求対応手続の具体化。

## 8. セキュリティ

- TLS 1.2/1.3 による通信暗号化
- Cloud Armor WAF による不正アクセス防止
- PII 自動検出・ブロック (多段防御)
- 出力サニタイザーによる間接プロンプトインジェクション対策
- 監査ログ 7 年 WORM 保存
