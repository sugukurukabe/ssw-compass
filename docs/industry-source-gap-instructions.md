# 16分野 公式資料・様式 収集指示書

> 作成日: 2026-04-30
> 目的: SSW Compass の `source-index.jsonl` / `forms-catalog.jsonl` / Vertex RAG v2 用に、16分野ごとの公式情報源と第3表・参考様式を補完する。

## 収集ルール

1. **一次情報のみ**: `moj.go.jp`, `mhlw.go.jp`, `mlit.go.jp`, `maff.go.jp`, `meti.go.jp` 等の政府・所管省庁ドメインだけ。
2. **URL は canonical**: 検索結果ページ、クエリ付き URL、PDF viewer URL、SNS redirect URL は不可。
3. **必要な情報**:
   - 分野別トップページ
   - 分野別運用要領 / 分野別運用方針
   - 第3表の公式 Excel/ページ
   - 分野別参考様式
   - 試験・評価試験の公式ページ
   - 協議会加入・誓約書・追加要件の公式ページ
4. **URL が 403/404 になる場合**:
   - 直リンクではなく、上位の安定ページを採用する。
   - `notes` に「direct file URL is blocked/unstable; use landing page」と書く。
5. **形式**: 下記テンプレートに沿って Slack / Markdown / CSV のいずれかで渡してください。

## 分野別の具体的な収集指示

起点ページ:

- 入管の分野別リンク集: `https://www.moj.go.jp/isa/applications/ssw/nyuukokukanri01_00130.html`
- 漁業: `https://www.jfa.maff.go.jp/j/kikaku/tokuteiginou.html`

全分野で共通してほしいもの:

1. **分野別トップページ**: 入管の「分野別所管省庁ウェブサイトリンク」から飛んだ先の安定 URL。
2. **分野別運用方針**: タイトルに「制度の運用に関する方針」が入る PDF または掲載ページ。
3. **分野別運用要領**: タイトルに「運用要領」「運用要領別冊」が入る PDF または掲載ページ。
4. **第3表 / 提出書類一覧**: 入管の特定技能申請ページにある分野別「第3表」。Excel 直リンクが不安定なら安定ページでよい。
5. **分野別参考様式**: 「分野参考様式第○号」「誓約書」「協議会加入証明」「実務経験証明」など。
6. **試験ページ**: 技能測定試験、評価試験、2号試験、受験手続、合格証明書発行ページ。
7. **協議会・追加要件**: 協議会加入、受入れ機関認定、許可・免許・講習・研修・実務経験など、分野固有の要件ページ。
