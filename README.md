# Visa Compass Japan (VCJ)

> *The compass for Japanese visa procedures.*
> 「日本ビザ手続きの羅針盤。」
> *Kompas untuk prosedur visa Jepang.*

Visa Compass Japan grounds your Japanese Specified Skilled Worker visa
questions in 出入国在留管理庁 official sources. We point the way —
gyoseishoshi handle the rest.

特定技能ビザの公式情報を、出入国在留管理庁ソースに直接接続。
羅針盤の役割に徹し、申請代理は行政書士へ。

Visa Compass Japan menghubungkan pertanyaan visa Pekerja Berketerampilan
Khusus Jepang Anda langsung ke sumber resmi 出入国在留管理庁.
Kami menunjukkan arah — gyoseishoshi menangani sisanya.
